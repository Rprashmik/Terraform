# Terraform AWS Infrastructure

This repository contains Terraform code to provision AWS infrastructure.

## Resources Created
- EC2 Instance
- Security Group
- Key Pair
- User Data Script

## Features
- Modular Terraform structure
- Remote backend with S3 and DynamoDB
- Environment-based deployment (dev/prod)

## Usage
terraform init
terraform plan
terraform apply

## Tools Used
- Terraform
- AWS
- Jenkins (CI/CD)
